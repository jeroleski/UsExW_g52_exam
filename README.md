# UsExW_g52_exam

## About the project

This car sharing website, were created as part of the course
'User experience og webprogrammering (Spring 2022)' at ITU.

## Running the project

#### Set up dependencies:

``$ npm install -g firebase-tools``

#### Run the project:

``$ firebase serve``

#### Visit the website at:

``http://localhost:5000``

### Close the server:

``Ctrl + C``
